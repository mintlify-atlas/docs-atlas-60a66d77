## What is Gist Web SDK?

The Gist Web SDK enables you to deliver rich, interactive in-app messages to your web application. Build beautiful flows with no code in Customer.io and deliver them instantly to your users.

With Gist, you can create:

- **Modal messages** - Attention-grabbing popups and overlays
- **Embedded messages** - Seamlessly integrated banners and notices
- **In-app inbox** - A persistent message center for your users
- **Multi-step flows** - Interactive onboarding and feature tours
- **Targeted messages** - Route-based and user-specific content

## How It Works

1. **Initialize** the SDK with your Customer.io site ID and configuration
2. **Identify** users with tokens or use anonymous sessions
3. **Set routes** to enable route-based message targeting
4. **Listen** to events to respond to user interactions
5. **Show messages** programmatically or let them trigger automatically

<Note>
  Gist Web SDK works seamlessly with Customer.io's message builder. Create your messages in the Customer.io dashboard and the SDK handles delivery and display.
</Note>
